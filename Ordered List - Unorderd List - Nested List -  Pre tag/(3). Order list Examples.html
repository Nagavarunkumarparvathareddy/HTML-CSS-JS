Observe A,B,C,D ordering

What is HTML ?

  1. HyperText Markup Language, used for creating web pages.
  2. A programming language used for developing applications.
  3. A database management system for storing data.
  4. A file format for images and videos.

What is CSS ?

  1. Cascading Style Sheets, a language used for styling web pages.
  2. A markup language for creating web applications.
  3. A database management system for storing data.
  4. A programming language used for server-side scripting.

What is Javascript ?

  1. A high-level programming language primarily used for creating interactive web pages.
  2. A markup language used for structuring web content.
  3. A database management system for storing large datasets.
  4. A file format used for storing multimedia content
